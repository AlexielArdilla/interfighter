# interfighter

Este libro describe un proyecto de software para optimizar el trafico de datos en Internet, además de proponer ideas para la programación de una aplicación principal (Interfighter). Es un libro para desarrolladores y fanáticos de la informática. La descripción del lenguaje de programación orientado a los ideogramas como objetos es una de las principales atracciones del libro.
